
Vaayu
======

.. only:: html

   :Version: |release|
   :Date: |today|

Vaayu is a python library that provides various utilities for use in Wind Energy
analysis workflows created with an intent to automate various tasks,
specifically setup, pre-processing, and post-processing. This software is
licensed under `Apache License Version 2.0
<http://www.apache.org/licenses/LICENSE-2.0>`_ open-source license.

Vaayu documentation is divided into two parts:

:ref:`user_guide`

   This part provides detailed information regarding the features and
   capabilities of the python library, as well as installation instructions. New
   users should start here to familiarize themselves with the library.

:ref:`dev_guide`

   The developer guide provides comprehensive documentation of the Vaayu API and
   is directed towards users interested in building their own scripts on top of
   the core functionality present within Vaayu. Where appropriate, the API
   docuementation is interspersed with discussions on the overall design and
   relationship between the various packages and modules.
